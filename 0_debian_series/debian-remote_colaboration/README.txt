## Stream Desktop with the OBS Project

For streaming desktop, see the OBS project.  It's cross platform and and can stream high fps games like Left 4 Dead.  

See https://obsproject.com/download#linux for these instructions:

```
sudo add-apt-repository ppa:kirillshkrogalev/ffmpeg-next && \
  sudo apt-get update && sudo apt-get install ffmpeg 


sudo add-apt-repository ppa:obsproject/obs-studio && \
  sudo apt-get update && sudo apt-get install obs-studio
```
